# 关于数组初始化
` int a[5] = { 0 };    // 全部初始化为０ `

全部初始化为０的那行代码确实是没问题的，可以正常工作。问题就出在想把数组全部初始化成一个非０的数，即非默认值，是行不通的,例如：

` int a[5] = { 1 };    // 我想全部初始化为１ `

查看内存发现，只有数组的第一个元素被初始化为１，其他全为０

## **有一条基本的语法规则：**

**数组初始化列表中的元素个数小于指定的数组长度时，不足的元素补以默认值。**
